Programming Language Support

Code that builds on Linux will generally build on CircleCI 2.0. For some language versions, CircleCI provides demo applications with YAML file templates and instructions:

  • Android Language Guide

  • Clojure Language Guide (v1.2.0 and later)

  • Elixir Language Guide (v1.2 and later)

  • Go Language Guide (v1.7 and later)

  • Java Language Guide (Java 8 and later)

  • JavaScript Language Guide (Node.js 4 and later)

  • PHP Language Guide (PHP 5 and later)

  • Python Language Guide (Python 2 and later)

  • Ruby and Rails Guide (Ruby 2 and later)

Build projects in C, C#, C++, Clojure, Elixir, Erlang, Go, Groovy, Haskell, Haxe, Java, JavaScript, Node.js, Perl, PHP, Python, Ruby, Rust, Scala and many more.
